# Headphones Landing Page

## Description
A fully responsive landing page for a headphones company, implemented from a Figma design without using any CSS frameworks or JavaScript.

## Project Overview
This project demonstrates proficiency in:
- Semantic HTML5
- CSS3 (Flexbox, Grid, responsive design)
- Mobile-first approach
- Accessibility best practices
- Pixel-perfect implementation from design mockups

## Design
The design was provided via Figma and includes:
- Desktop layout (1000px max width)
- Mobile layout (480px and below)
- Custom fonts: Source Sans Pro and Spin Cycle OT
- Specific hover states and interactions

## Features
- Fully responsive design
- Mobile breakpoint at 480px
- Custom hover effects on links and buttons
- Accessible markup
- Cross-browser compatible

## Technologies Used
- HTML5
- CSS3
- No frameworks or libraries
